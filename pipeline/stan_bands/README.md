Stan bands support data
=======================

This directory contains the wavelength-dependent atmospheric cross sections
used for the repeated Stan-band N2 classes.

Source URLs:

- https://soho.nascom.nasa.gov/solarsoft/sdo/eve/cal_data/cross_sections.dat

The key field for this pipeline is molecular nitrogen, column 3 in
`cross_sections.dat`, in megabarns. The official Stan-band logic separates
repeated intervals using 4 and 31 megabarn thresholds.
